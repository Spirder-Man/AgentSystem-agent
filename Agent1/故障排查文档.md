## ⭐⭐⭐ 高频错误 #1：工具调用判断逻辑缺陷

### 错误现象

**输入：**

```
查看工业传感参数是否有异常
```

**阶段1输出：**

```
思考: 需要查看工业传感的实时温度并与安全阈值比较
需要工具: GetSpindleTemperature, GetTemperatureThreshold
工具列表: GetSpindleTemperature,GetTemperatureThreshold
```

**阶段2错误输出：**

```
✅ 不需要调用工具
```

***

### 错误位置

**文件：** `SmartDialogSystem.cs`

**函数：** `ParseToolPlan(string analysis)`

**错误代码：**

```csharp
if (trimmed.StartsWith("需要工具:"))
{
    var value = trimmed.Substring("需要工具:".Length).Trim();
    plan.NeedsTools = value.Contains("是");  // ❌ 只检查"是"字！
}
```

***

### 错误原因分析

| 原因       | 说明                                                                |
| -------- | ----------------------------------------------------------------- |
| \*\*假设错误 | 开发者假设模型一定会输出"是"或"否"                                               |
| \*\*实际输出 | 模型直接输出工具列表：`需要工具: GetSpindleTemperature, GetTemperatureThreshold` |
| \*\*判断逻辑 | `value.Contains("是")` 为 false → `NeedsTools = false`              |

***

### 修复方案

```csharp
if (trimmed.StartsWith("需要工具:"))
{
    var value = trimmed.Substring("需要工具:".Length).Trim();

    // ✅ 修复1：支持"是"字，也支持直接是工具列表
    plan.NeedsTools = value.Contains("是") || !string.IsNullOrWhiteSpace(value);

    // ✅ 修复2：如果"需要工具"后面直接是工具列表，也解析
    if (!value.Contains("是") && !value.Contains("否") && !string.IsNullOrWhiteSpace(value))
    {
        plan.ToolNames = value.Split(',')
                              .Select(t => t.Trim())
                              .Where(t => !string.IsNullOrWhiteSpace(t) &&
                                         !t.Contains("无") &&
                                         !t.Contains("空"))
                              .ToList();
        plan.NeedsTools = plan.ToolNames.Count > 0;
    }
}

// ✅ 修复3：只要工具列表有内容，强制设置NeedsTools=true
if (plan.ToolNames.Count > 0)
    plan.NeedsTools = true;
```

## ⭐⭐ 高频错误 #2：工具名不匹配

### 错误现象

**模型输出：**

```
工具列表: CurentTime
```

**工具调用失败：**

```
未知工具: CurentTime
```

***

### 错误位置

**文件：** `SmartDialogSystem.cs`

**函数：** `CallToolAsync(string toolName)`

**错误代码：**

```csharp
return toolName switch
{
    "GetSpindleTemperature" => _tools.GetSpindleTemperature(),
    "GetTemperatureThreshold" => _tools.GetTemperatureThreshold(),
    "GetCurrentTime" => _tools.GetCurrentTime(),
    "GetSpindleTemperature()" => _tools.GetSpindleTemperature(),
    "GetTemperatureThreshold()" => _tools.GetTemperatureThreshold(),
    "GetCurrentTime()" => _tools.GetCurrentTime(),
    // ❌ 缺少拼写容错！
    _ => $"未知工具: {toolName}"
};
```

***

### 错误原因分析

| 原因         | 说明                                     |
| ---------- | -------------------------------------- |
| \*\*模型拼写错误 | `GetCurrentTime` → `CurentTime`（少了一个r） |
| \*\*严格匹配   | switch 语句只匹配完全相同的字符串                   |
| \*\*无法容错   | 轻微拼写差异就调用失败                            |

***

### 修复方案

```csharp
private async Task<string> CallToolAsync(string toolName)
{
    toolName = toolName.Trim();
    Console.WriteLine($"    🔧 尝试调用: {toolName}");

    // ✅ 修复1：模糊匹配（优先）
    var lowerTool = toolName.ToLower();

    if (lowerTool.Contains("spindle") && lowerTool.Contains("temperature"))
        return _tools.GetSpindleTemperature();
    if (lowerTool.Contains("threshold"))
        return _tools.GetTemperatureThreshold();
    if (lowerTool.Contains("time") && lowerTool.Contains("current"))
        return _tools.GetCurrentTime();
    if (lowerTool.Contains("calculate") || lowerTool.Contains("计算"))
    {
        var idx = toolName.IndexOf('(');
        if (idx > 0)
        {
            var arg = toolName.Substring(idx + 1).TrimEnd(')');
            return _tools.Calculate(arg);
        }
    }

    // ✅ 修复2：精确匹配（兜底）
    if (toolName.Contains("("))
    {
        var openParen = toolName.IndexOf('(');
        var closeParen = toolName.IndexOf(')');
        if (closeParen == -1) closeParen = toolName.Length;
        var name = toolName.Substring(0, openParen).Trim();
        var arg = toolName.Substring(openParen + 1, closeParen - openParen - 1).Trim();

        if (name == "Calculate")
            return _tools.Calculate(arg);
        if (name == "WebSearch")
            return await _tools.WebSearch(arg);
    }

    return toolName switch
    {
        "GetSpindleTemperature" => _tools.GetSpindleTemperature(),
        "GetTemperatureThreshold" => _tools.GetTemperatureThreshold(),
        "GetCurrentTime" => _tools.GetCurrentTime(),
        "GetSpindleTemperature()" => _tools.GetSpindleTemperature(),
        "GetTemperatureThreshold()" => _tools.GetTemperatureThreshold(),
        "GetCurrentTime()" => _tools.GetCurrentTime(),
        "CurentTime" => _tools.GetCurrentTime(), // ✅ 修复3：常见拼写错误
        _ => $"未知工具: {toolName}"
    };
}
```

## ⭐⭐⭐ 高频错误 #7：DeepSeek-R1输出内部思考显示给用户

### 错误现象

**从截图显示：**

```
【阶段1】分析问题与规划工具

<think>
好，现在我要分析用户是否需要调用工具...
</think>
需要工具: 否
```

**问题：** `<think>` 标签及其内部内容直接输出给用户，这是DeepSeek-R1的推理过程，不应该展示给最终用户看。

***

### 错误位置

**文件：** `SmartDialogSystem.cs`

**函数：** `InvokeStreamAsync(string prompt, ConsoleColor color)`

**错误代码：**

```csharp
private async Task<string> InvokeStreamAsync(string prompt, ConsoleColor color)
{
    var result = new StringBuilder();
    Console.WriteLine();
    Console.ForegroundColor = color;

    try
    {
        await foreach (var chunk in _kernel.InvokePromptStreamingAsync(...))
        {
            result.Append(chunk);
            Console.Write(chunk);  // ❌ 直接输出，包含<think>
            await Task.Delay(15);
        }
    }
    catch (Exception ex)
    {
        Console.WriteLine($"\n⚠️ 生成时出错: {ex.Message}");
    }

    Console.ResetColor();
    Console.WriteLine();
    return result.ToString();
}
```

***

### 错误原因分析

| 原因             | 说明                                  |
| -------------- | ----------------------------------- |
| \*\*DeepSeek特性 | DeepSeek-R1是推理时会把思考过程放在`<think>`标签内 |
| \*\*直接输出       | 流式输出时没有过滤标签内容，导致整个输出全部输出给用户看        |
| \*\*用户体验差      | 用户不需要看到模型的内部推理过程，只需要最终答案            |

***

### 修复方案

```csharp
private async Task<string> InvokeStreamAsync(string prompt, ConsoleColor color)
{
    var result = new StringBuilder();
    Console.WriteLine();
    Console.ForegroundColor = color;

    // ✅ 修复：用于跟踪是否在think标签内
    bool isInThinkBlock = false;
    string buffer = "";

    try
    {
        await foreach (var chunk in _kernel.InvokePromptStreamingAsync<string>(prompt, cancellationToken: _cts.Token))
        {
            buffer += chunk;

            // 检测think标签并过滤
            while (true)
            {
                // 1. 检查是否有 <think> 开始标签
                int thinkStart = buffer.IndexOf("<think>");
                // 2. 检查是否有 </think> 结束标签
                int thinkEnd = buffer.IndexOf("</think>");

                if (thinkStart >= 0 && (thinkEnd < 0 || thinkStart < thinkEnd))
                {
                    // 在进入think块之前，先输出thinkStart之前的内容
                    string beforeThink = buffer.Substring(0, thinkStart);
                    if (!string.IsNullOrEmpty(beforeThink))
                    {
                        result.Append(beforeThink);
                        Console.Write(beforeThink);
                    }

                    isInThinkBlock = true;
                    buffer = buffer.Substring(thinkStart + "<think>".Length);
                }
                else if (isInThinkBlock && thinkEnd >= 0)
                {
                    // 结束think块，跳过中间内容
                    buffer = buffer.Substring(thinkEnd + "</think>".Length);
                    isInThinkBlock = false;
                }
                else
                {
                    // 没有标签需要处理，退出循环
                    break;
                }
            }

            // 如果不在think块内，直接输出缓冲区剩余内容
            if (!isInThinkBlock && buffer.Length > 0)
            {
                result.Append(buffer);
                Console.Write(buffer);
                buffer = "";
            }

            await Task.Delay(15);
        }

        // 最后处理剩余的缓冲区
        if (!isInThinkBlock && buffer.Length > 0)
        {
            result.Append(buffer);
            Console.Write(buffer);
        }
    }
    catch (Exception ex)
    {
        Console.WriteLine($"\n⚠️ 生成时出错: {ex.Message}");
    }

    Console.ResetColor();
    Console.WriteLine();
    return result.ToString();
}
```

## ⭐⭐⭐ 高频错误 #9: DeepSeek-R1调用被取消/超时

### 错误现象

**错误输出：**

```
⚠️ 生成时出错: The invocation of function 'InvokePromptStreamingAsync_xxxx' was canceled.
```

**发生位置：**
通常发生在阶段6（生成最终结论），因为它的Prompt最长。

### 错误位置

**文件：** `SmartDialogSystem.cs`

**函数：** `InvokeStreamAsync` 和 `GenerateFinalConclusionAsync`

### 错误原因分析

| 原因           | 说明                                 |
| ------------ | ---------------------------------- |
| **Prompt太长** | 阶段6的Prompt包含历史+工具数据+初步结论+反思结果，内容很多 |
| **处理时间长**    | DeepSeek-R1推理需要时间，容易超时             |
| **网络连接**     | 本地Ollama服务偶尔会有连接问题                 |
| **没有重试**     | 之前代码超时后直接失败，没有重试机制                 |

### 修复方案

**修复1：添加重试机制**

```csharp
private const int MaxRetries = 3; // 最大重试次数
private const int RetryDelayMs = 1000; // 重试延迟（毫秒）

/// <summary>
/// 带有重试机制的流式调用函数
/// </summary>
private async Task<string> InvokeStreamWithRetryAsync(string prompt, ConsoleColor color, string stageName = "")
{
    Exception lastException = null;
    
    for (int attempt = 1; attempt <= MaxRetries; attempt++)
    {
        try
        {
            if (attempt > 1)
            {
                Console.WriteLine($"\n🔄 第{attempt}次重试 {stageName}...");
            }
            
            return await InvokeStreamAsync(prompt, color);
        }
        catch (OperationCanceledException ex)
        {
            lastException = ex;
            Console.WriteLine($"\n⏰ 请求超时 ({attempt}/{MaxRetries}): {ex.Message}");
            
            if (attempt < MaxRetries)
            {
                await Task.Delay(RetryDelayMs);
            }
        }
        catch (Exception ex)
        {
            lastException = ex;
            Console.WriteLine($"\n❌ 发生错误 ({attempt}/{MaxRetries}): {ex.Message}");
            
            if (attempt < MaxRetries)
            {
                await Task.Delay(RetryDelayMs);
            }
        }
    }
    
    // 所有重试都失败了
    Console.ForegroundColor = ConsoleColor.Red;
    Console.WriteLine($"\n❌ 所有重试都失败了: {lastException?.Message}");
    Console.ResetColor();
    
    return $"生成失败: {lastException?.Message}";
}
```

**修复2：优化阶段6的Prompt**

```csharp
// 优化前的Prompt - 太长了
var prompt = $@"{systemPrompt}

{toc}
```

```csharp
        return content;

    // 移除 <think> 标记
    content = content.Replace("<think>", "").Replace("</think>", "");

    // 移除行首编号格式（如 "5. "）
    content = System.Text.RegularExpressions.Regex.Replace(content, @"^\d+\.\s*", "", System.Text.RegularExpressions.RegexOptions.Multiline);

    content = System.Text.RegularExpressions.Regex.Replace(content, @"\n\s*\n", "\n");

    content = content.Trim();
```
