# 手动测试指南

## 快速开始

### 1. 编译项目
```bash
dotnet build --configuration Release
```

### 2. 运行程序
```bash
dotnet run --configuration Release
```

## 测试流程

### 第一步：测试模块9 - Smart Auto Router

#### 测试1：简单对话模式
```
选择：9
输入：你好，我叫测试员，你叫小助手
预期：⚡ 识别为：简单对话模式

输入：我是谁？
预期：🧠 从记忆中找到答案！ 你叫测试员

输入：你叫什么？
预期：🧠 从记忆中找到答案！ 我叫小助手

输入：exit
```

#### 测试2：专业诊断模式
```
选择：9
输入：检查机床主轴温度
预期：📋 识别为：专业诊断模式
预期：【分析中】→【调用工具】→【生成答案】

输入：exit
```

#### 测试3：模式切换
```
选择：9
输入：你好，我叫李工
→ 简单对话

输入：检查温度
→ 切换到专业诊断

输入：谢谢
→ 切换回简单对话

输入：我是谁？
→ 记忆应该记得"李工"

输入：exit
```
