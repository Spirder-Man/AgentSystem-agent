# 工业 AI Agent - 手动测试指南

## 快速开始

### 1. 编译项目
```bash
dotnet build --configuration Release
```

### 2. 运行程序
```bash
dotnet run --configuration Release
# 或直接运行编译后的 EXE
.\bin\Release\net8.0\Agent1.exe
```

## 检测四大范式调用的方法

### 方法1：代码检查
打开以下文件查看：
- `Agent1/Modules/SmartAutoRouterModule.cs` → 没有调用CoT/ReAct/Reflection/RAG
- `Agent1/IndustrialDiagnosticHandler.cs` → 使用ToolService，不调用范式
- `Agent1/SmartDialogSystem.cs` → 独立实现，不使用四大范式

### 方法2：运行时观察
运行模块8或9时，**不会看到**：
- "CoT"、"思维链" 等字样
- "ReAct"、"推理-行动" 等字样  
- "Reflection"、"反思" 等字样
- "RAG"、"检索增强" 等字样

### 方法3：与模块1-6对比
选择模块1（CoT）或模块3（ReAct），可以看到明显不同的输出格式
