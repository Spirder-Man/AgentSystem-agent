# 第一阶段代码初步理解

## 一、核心架构关系

### 1.1 架构图

```
Program.cs
    │
    ▼
ModuleDispatcher.cs (模块调度器 - 指挥中心)
    │
    ▼
ModuleFactory.cs (模块工厂 - 创建模块)
    │
    ▼
IInferenceModule.cs (接口)
    │
    ├── CoTSolidModule
    ├── CoTStreamModule
    ├── ReActSolidModule
    └── ...
```

### 1.2 各角色职责

| 组件 | 职责 |
|------|------|
| Program.cs | 主程序入口 |
| ModuleDispatcher | 模块调度器，指挥中心 |
| ModuleFactory | 模块工厂，根据类型创建模块 |
| IInferenceModule | 模块接口定义 |
| CoT/ReAct/RAG等 | 具体模块实现 |
