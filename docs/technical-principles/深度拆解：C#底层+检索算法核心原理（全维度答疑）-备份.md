# 深度拆解：C#底层+检索算法核心原理（备份版本）

## 一、C# 内存基础

### 1.1 堆与栈的区别

- **栈**：存储值类型和引用类型的引用（地址），访问速度快
- **堆**：存储引用类型的实际数据，容量大

### 1.2 常用集合类型

|集合|底层结构|特性|适用场景|
|---|---|---|---|
|List<T>|动态数组|有序、可重复|有序列表存储|
|Dictionary<TKey,TValue>|哈希表|键值对映射|O(1)查找|
|HashSet<T>|哈希表|元素唯一|去重|
|SortedList<TKey,TValue>|有序数组|键有序|需要排序的场景|

## 二、倒排索引原理

### 2.1 核心数据结构

```csharp
Dictionary<string, Dictionary<int, int>>
// 外层key: 分词
// 内层: 文档ID -> 词频
```

### 2.2 查询流程

1. 对查询进行分词
2. 查找每个分词对应的文档列表
3. 合并去重得到候选文档集
4. 计算BM25得分

## 三、BM25算法

### 3.1 公式

```csharp
score = Σ IDF(token) × (tf × (K1+1)) / (tf + K1 × (1-B+B×len/avgLen))
```

### 3.2 参数说明

|参数|作用|建议值|
|---|---|---|
|K1|词频饱和度|1.2-1.5|
|B|文档长度归一化|0.75|
