## 核心错误记录

### 错误1：向量扩展未安装

**现象**:
```
ERROR:  operator does not exist: vector <-> vector
LINE 1: SELECT embedding <-> '[0.1,0.2,0.3]'::vector FROM ...
```

**修复方案**:
```sql
CREATE EXTENSION IF NOT EXISTS vector;
CREATE INDEX idx_chemical_embedding ON chemical_documents USING hnsw (embedding vector_cosine_ops);
```

## 向量数据库配置

### 1. PostgreSQL + pgvector 安装

```bash
# Ubuntu/Debian
apt-get install postgresql-16 postgresql-16-pgvector

# 验证安装
psql -c "SELECT * FROM pg_extension WHERE extname = 'vector';"
```

### 2. 表结构设计

```sql
CREATE TABLE chemical_documents (
    id SERIAL PRIMARY KEY,
    content TEXT NOT NULL,
    embedding vector(768),
    regulation_type VARCHAR(100),
    priority INTEGER,
    source_file VARCHAR(255),
    chemical_type VARCHAR(100),
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

### 3. 索引优化

```sql
-- HNSW索引优化检索性能
CREATE INDEX idx_chemical_embedding ON chemical_documents USING hnsw (embedding vector_cosine_ops);
```
