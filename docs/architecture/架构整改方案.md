# Services 文件夹架构整改方案

## 一、问题分析

### 1.1 硬编码问题统计

| 文件 | 问题类型 | 严重程度 | 说明 |
|------|---------|---------|------|
| **ModelConfig.cs** | 配置硬编码 | 🟥 高 | 模型ID、Endpoint都是const硬编码 |
| **LlmService.cs** | 重试参数硬编码 | 🟨 中 | MaxRetries、RetryDelayMs 是常量 |
| **LlmService.cs** | Buffer阈值硬编码 | 🟨 中 | bufferFlushThreshold = 50 |
| **MemoryService.cs** | 关键词硬编码 | 🟥 高 | 大量if-else包含"我叫"、"温度"等硬编码 |
| **MemoryService.cs** | 正则表达式硬编码 | 🟨 中 | 温度匹配、阈值匹配的正则写死 |
| **ToolService.cs** | 工具名称硬编码 | 🟥 高 | GetSpindleTemperature等工具名硬编码 |
| **ToolService.cs** | 工具描述硬编码 | 🟨 中 | GetToolDescriptions()返回的内容写死 |
| **SessionService.cs** | 静态类依赖 | 🟥 高 | 直接调用SessionManager静态方法，紧耦合 |

## 二、架构设计原则

### 2.1 本次整改遵循的原则

| 原则 | 说明 |
|------|------|
| **依赖倒置 (DIP)** | 依赖抽象不依赖具体实现 |
| **开闭原则 (OCP)** | 对扩展开放，对修改关闭 |
| **单一职责 (SRP)** | 每个类只做一件事 |
| **配置外部化** | 所有配置从代码中移出 |
| **可测试性** | 易于单元测试和Mock |

### 3.3 整改：MemoryService.cs

#### 问题点修复
1. ✅ 移除所有if-else硬编码关键词
2. ✅ 所有配置从MemoryConfig读取
3. ✅ 使用策略模式简化逻辑

```csharp
namespace Agent1.Services
{
    public class MemoryService : IMemoryService
    {
        private readonly UserProfile _userProfile;
        private readonly Dictionary<string, string> _keyFacts;
        private readonly MemoryConfig _config;
        
        public MemoryService(MemoryConfig config)
        {
            _config = config;
            _userProfile = new UserProfile();
            _keyFacts = new Dictionary<string, string>();
        }
        
        public string TryAnswerFromMemory(string userInput)
        {
            var lower = userInput.ToLower();
            
            // 使用配置的关键词
            if (MatchesAny(lower, _config.UserNameKeywords))
            {
                if (!string.IsNullOrEmpty(_userProfile.UserName))
                {
                    if (!string.IsNullOrEmpty(_userProfile.AssistantName))
                    {
                        return $"你叫 {_userProfile.UserName}，我叫 {_userProfile.AssistantName}！";
                    }
                    return $"你叫 {_userProfile.UserName}！";
                }
            }
            
            if (MatchesAny(lower, _config.AssistantNameKeywords))
            {
                if (!string.IsNullOrEmpty(_userProfile.AssistantName))
                {
                    if (!string.IsNullOrEmpty(_userProfile.UserName))
                    {
                        return $"我叫 {_userProfile.AssistantName}，你叫 {_userProfile.UserName}！";
                    }
                    return $"我叫 {_userProfile.AssistantName}！";
                }
            }
            
            // 使用配置的温度关键词
            if (MatchesAny(lower, _config.TimeKeywords) && lower.Contains("温度"))
            {
                if (_keyFacts.ContainsKey("主轴温度"))
                {
                    return $"你好！之前检查的温度是 {_keyFacts["主轴温度"]}。还有什么需要我帮你的吗？";
                }
            }
            
            if (MatchesAny(lower, _config.TemperatureKeywords))
            {
                if (_keyFacts.ContainsKey("安全阈值"))
                {
                    return $"安全阈值是 {_keyFacts["安全阈值"]}。";
                }
            }
            
            return null;
        }
        
        public void ExtractAndStoreKeyFacts(string userInput, string assistantResponse)
        {
            var lowerInput = userInput.ToLower();
            var lowerResponse = assistantResponse.ToLower();
            
            // 提取用户名（改进：减少硬编码）
            foreach (var keyword in _config.UserNameKeywords)
            {
                var idx = lowerInput.IndexOf(keyword);
                if (idx >= 0)
                {
                    var nameStart = idx + keyword.Length;
                    if (nameStart < userInput.Length)
                    {
                        var nameEnd = FindNameEnd(userInput, nameStart);
                        var name = userInput.Substring(nameStart, nameEnd - nameStart).Trim();
                        if (!string.IsNullOrWhiteSpace(name))
                        {
                            _userProfile.UserName = name;
                        }
                    }
                    break;
                }
            }
            
            // 提取温度（使用配置的正则）
            if (lowerResponse.Contains("温度"))
            {
                var match = System.Text.RegularExpressions.Regex.Match(assistantResponse, _config.TemperatureRegex);
                if (match.Success)
                {
                    _keyFacts["主轴温度"] = match.Value;
                }
            }
            
            if (lowerResponse.Contains("阈值"))
            {
                var match = System.Text.RegularExpressions.Regex.Match(assistantResponse, _config.ThresholdRegex);
                if (match.Success)
                {
                    _keyFacts["安全阈值"] = match.Value;
                }
            }
        }
        
        // 辅助方法
        private bool MatchesAny(string input, List<string> keywords)
        {
            return keywords.Any(k => input.Contains(k));
        }
        
        private int FindNameEnd(string input, int start)
        {
            var delimiters = new[] { "，", "。", " ", ",", "." };
            var minEnd = input.Length;
            
            foreach (var delimiter in delimiters)
            {
                var idx = input.IndexOf(delimiter, start);
                if (idx >= 0 && idx < minEnd)
                    minEnd = idx;
            }
            
            return minEnd;
        }
    }
}
```

### 3.4 整改：ToolService.cs

#### 问题点修复
1. ✅ 工具列表从配置读取
2. ✅ 工具名称判断使用配置，不是硬编码

```csharp
namespace Agent1.Services
{
    public class ToolService : IToolService
    {
        private readonly IndustrialTools _tools;
        private readonly ILlmService _llmService;
        private readonly ToolConfig _config;
        
        public ToolService(ILlmService llmService, ToolConfig config)
        {
            _tools = new IndustrialTools();
            _llmService = llmService;
            _config = config;
        }
        
        public async Task<ToolPlan> AnalyzeAndPlanToolsAsync(string userInput, string history)
        {
            var plan = new ToolPlan();
            var lowerInput = userInput.ToLower();
            
            foreach (var tool in _config.Tools)
            {
                if (tool.KeywordTriggers.Any(kw => lowerInput.Contains(kw.ToLower())))
                {
                    plan.NeedsTools = true;
                    plan.ToolNames.Add(tool.Name);
                    Console.WriteLine($"\n🤔 分析中... 检测到{tool.Name}相关问题");
                    break;
                }
            }
            
            if (!plan.NeedsTools)
            {
                plan.NeedsTools = false;
                Console.WriteLine("\n🤔 分析中... 不需要调用工具");
            }
            
            return plan;
        }
        
        private string GetToolDescriptions()
        {
            var sb = new System.Text.StringBuilder();
            for (int i = 0; i < _config.Tools.Count; i++)
            {
                sb.AppendLine($"{i + 1}. {_config.Tools[i].Name} - {_config.Tools[i].Description}");
            }
            return sb.ToString();
        }
    }
}
```

### 3.5 整改：SessionService.cs - 解耦静态依赖

#### 问题
当前SessionService直接调用SessionManager静态方法，紧耦合！

#### 方案1：创建接口（推荐）
```csharp
namespace Agent1.Services
{
    public interface ISessionManager
    {
        SessionContext CreateSession(string? id, SessionType type);
        SessionContext? GetSession(string sessionId);
        void AddDialogTurn(string sessionId, string role, string content);
        void ClearDialogHistory(string sessionId);
        string GetFormattedHistory(string sessionId, int maxTurns);
        string GetContextSummary(string sessionId, int maxTurns);
        int GetHistoryCount(string sessionId);
    }
    
    public class SessionManagerAdapter : ISessionManager
    {
        public SessionContext CreateSession(string? id, SessionType type)
            => SessionManager.CreateSession(id, type);
            
        // ... 其他方法同理
    }
    
    public class SessionService : ISessionService
    {
        private readonly ISessionManager _sessionManager;
        
        public SessionService(ISessionManager sessionManager)
        {
            _sessionManager = sessionManager;
        }
        
        // 所有方法通过_sessionManager调用
    }
}
```
